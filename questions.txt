CRUD stands for CREATE, READ, UPDATE, DELETE. In association with HTTP method its POST, GET, PUT, and DELETE.

Question 4: What do HTTP 1XX Status Codes mean? HTTP 2xx? HTTP 3xx? HTTP 4xx? HTTP 5xx?
1xx mean request was recevied, continuing process. 2xx means reuqest was successful. 3xx means redirection, 
4xx means client errors. 5xx means server errors.
